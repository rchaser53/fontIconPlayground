# fontIconPlayground
## setup

1. create "font.config.json" file. see configuration
2. open command line
3. $ npm install webfont-icons-generator -g
4. $ webfont-icons-generator

## configuration
like below

``` font.config.json
{
  "src": "./img/*.svg",
  "pwd": "./img",
  "fontName": "originalFont",
  "dist": "./fonts"
}
```
